## src/内の現状

### 現在の部品化状況
- **components/** : ui.js のみ（UIコンポーネント不足）
- **services/** : auth.js, database.js, logger.js（基本サービスのみ）
- **features/** : main.js のみ（アプリ固有機能）
- **custom/** : app-config.js, styles.css（カスタマイズ領域）
- **scripts/** : 起動スクリプト類
- **examples/** : サンプルコード
- **core-legacy/** : 旧バージョン参照用
